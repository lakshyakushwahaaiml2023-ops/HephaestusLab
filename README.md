# HephaestusLab

HephaestusLab is a growing collection of interactive physics simulations.

Right now, this repository contains the following experiments:
- **ChaoticDoublePendulum**: Single and chaotic double pendulum simulations.
- **ClothSimulation**: A Position Based Dynamics (PBD) cloth simulation with wind, sphere collision, and Phong shading.

## Goal

Build clear, visual, and educational physics demos that are easy to run and explore.

## What will be added later

More physics simulations will be added over time (for example, mechanics and dynamical systems projects).
